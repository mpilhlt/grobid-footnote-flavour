# Grobid-footnote-flavour

**Work in progress** 

Data and workflow for developing a new flavour for supporting documents from Law and Digital Humanities in [Grobid](https://github.com/kermitt2/grobid). 

## Data Structure

The structure of the data will be based on batches and will follow this structure:

```
├── batches
│   └── batch_1
│       ├── 0_input
│       │   ├── 10.12946__rg01__036-055.pdf
│       │   └── 10.5771__2699-1284-2020-1-16.pdf
│       ├── 1_generated
│       └── 2_corrected
```
